# Orbit Visualization Tool
## How to Use
1. Create python environment (used 3.13 last)
2. Install all packages in import in 'traj_gen.py'
3. Run python script 'traj_gen.py'
4. Add maneuvers into main method with 'traj_gen.py' or use default
5. open up orbit_vis.html
6. load json file with 'import json' button
7. View and control visualization with interface
